# ✨ Interactive Particle Cursor

A mesmerizing, interactive particle attraction cursor effect built with Three.js. This project transforms an ordinary cursor into a visually stunning, physics-based particle system.

🔗 **Live Demo:** [Check out the live cursor here!](https://pankajtiwari-art.github.io/interactive-particle-cursor/)

---

## 🌟 Features

* **Physics-Based Attraction:** Particles smoothly follow and orbit the cursor using Three.js logic.
* **🎨 Dynamic Random Colors:** Click a button to instantly randomize the color of the glowing particles.
* **🔄 Auto-Color Mode:** Toggle a built-in interval that automatically shifts particle colors every 1.5 seconds.
* **📏 Real-Time Size Adjustment:** Use a sleek slider to adjust the size of the particles smoothly without reloading.
* **💎 Glassmorphism UI:** Modern, floating, translucent control panel designed with CSS backdrop-filters.

---

## 🛠️ Tech Stack

* **HTML5:** Semantic layout and canvas generation.
* **CSS3:** Flexbox, Glassmorphism, CSS Transitions, and Custom Styling.
* **JavaScript (ES6 Module):** Interactive DOM manipulation and event handling.
* **Three.js Components:** The core 3D particle engine (`threejs-components`).

## 🚀 How to Run Locally

If you want to play with the code on your machine:

1. Clone this repository:
   ```bash
   git clone [https://github.com/pankajtiwari-art/interactive-particle-cursor.git](https://github.com/pankajtiwari-art/interactive-particle-cursor.git)
2. Open the project folder in VS Code.
3. Use the Live Server extension to run the project. (Note: Opening the index.html directly via file:// might block the module import due to strict CORS policies).
